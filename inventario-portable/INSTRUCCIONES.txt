🎯 SISTEMA DE INVENTARIO - VERSIÓN PORTABLE

📋 ¿QUÉ ES?
Esta es una aplicación de escritorio para gestión de inventario que NO necesita instalación.
Funciona directamente en tu computadora sin necesidad de programas adicionales.

🚀 CÓMO USARLA:

OPCIÓN 1: Doble clic (Windows)
1. Descarga la carpeta "inventario-portable"
2. Haz doble clic en el archivo "index.html"
3. Se abrirá en tu navegador predeterminado
4. ¡Listo! Ya puedes usar la aplicación

OPCIÓN 2: Script automático (Windows)
1. Haz doble clic en "INICIAR.bat"
2. La aplicación se abrirá automáticamente en pantalla completa
3. Sin necesidad de abrir manualmente el navegador

OPCIÓN 3: Manual (Cualquier sistema)
1. Abre tu navegador web (Chrome, Firefox, Edge, etc.)
2. Arrastra el archivo "index.html" a la ventana del navegador
3. O usa Archivo > Abrir > selecciona "index.html"

✨ CARACTERÍSTICAS:
✅ Crear productos con nombre, descripción, stock y precio
✅ Editar existencias (agregar o restar unidades)
✅ Ver estadísticas en tiempo real
✅ Alertas de stock bajo
✅ Eliminar productos
✅ Datos guardados localmente (se pierden si borras la carpeta)
✅ Funciona sin conexión a internet
✅ Interfaz profesional y moderna

⚙️ REQUISITOS:
- Windows, Mac o Linux
- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- No necesita instalación
- No necesita conexión a internet

💾 ALMACENAMIENTO:
- Los datos se guardan en el navegador (localStorage)
- Si borras los datos del navegador, se perderá el inventario
- Para respaldar: copia toda la carpeta "inventario-portable"

🔧 ATAJOS DE TECLADO:
- Ctrl + N: Nuevo producto
- Esc: Cerrar ventanas emergentes

📞 SOPORTE:
Esta aplicación es autocontenida y no requiere mantenimiento técnico.
Si tienes problemas, asegúrate de:
1. Usar un navegador actualizado
2. No mover el archivo "index.html" de su carpeta
3. Permitir que el navegador guarde datos locales

🎉 ¡DISFRUTA TU SISTEMA DE INVENTARIO!
